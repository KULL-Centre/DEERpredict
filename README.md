Overview
--------

A package for DEER prediction over molecular dynamics ensembles. Can be installed with pip.

Installation / Usage
--------------------

To install use pip:

.. code:: bash

    $ pip install DEERpredict


Or clone the repo:

.. code:: bash

    $ git clone https://github.com/joaommartins/deerpredict.git
    $ cd deepredict

    $ pip install -e .
