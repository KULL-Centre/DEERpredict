Overview
--------

A package for DEER, FRET and PRE prediction based on  molecular dynamics ensembles. Can be installed with pip.

Installation / Usage
--------------------

To install use pip:

```bash
  pip install DEERpredict
```

Or clone the repo:

```bash
  git clone https://github.com/KULL-Centre/DEERpredict.git
  cd deepredict

  pip install -e . 
```
