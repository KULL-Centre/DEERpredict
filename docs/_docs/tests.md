# Testing

Tests can be run using the test running tool pytest:

~~~ bash
cd DEERpredict
python -m pytest
~~~

Tests and supplementary examples are also made available as Jupyter Notebooks in the `tests/data` folder:
- `HIV-1PR/HIV-1PR.ipynb`
- `nanodisc/nanodisc.ipynb`
- `ACBP/ACBP.ipynb`
- `article.ipynb`
